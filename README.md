# Marketing Data Analysis | Python Capstone 🐍

### Project Overview
This project focuses on transforming raw marketing datasets into actionable insights. It demonstrates a complete data workflow, from initial cleaning to exploratory analysis.

### Key Technical Achievements
* **Data Cleaning & Preprocessing:** Automated the handling of missing values and standardized data formats using **Pandas** and **NumPy**.
* **Exploratory Data Analysis (EDA):** Performed correlation analysis and trend identification to understand customer behavior.
* **Feature Engineering:** Corrected data types (e.g., Age) to ensure statistical accuracy in reporting.


### Tools & Technologies
* **Language:** Python.
* **Libraries:** Pandas, NumPy.
* **Environment:** Jupyter Notebook.


### Results
* Identified key correlations between customer demographics and purchasing patterns.
* Reduced manual data preparation time through automated Python scripts.
